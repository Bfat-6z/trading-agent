  when `path.exists()` but parse fails, so callers fail-closed.

  + `ping=False` + reset `_spot=None` on repeated failure (or thread-bounded wrapper like `_bounded_get`).
- **R3 silent `except: pass`** everywhere — hides failures (21h-stale, dead-but-heartbeating observers).
- **R4 non-atomic state writes + no idempotency key + no singleton lock** → crash/2-instance double-book
  or equity-wipe. FIX: mirror `shadow_trade_evaluator` (idempotency key + tmp+os.replace).
- **R5 stale-cache-served-as-fresh** — no age check on caches; `stamped`=build-time keeps ticking.
  FIX: age-check + emit `price_source`/`price_age_s` + UI "STALE" badge.
